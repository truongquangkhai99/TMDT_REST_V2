Đồ án môn Thương mại điện tử (API)

* Spring framework
  * Spring Web Services
  * Spring security
  * Spring boot
  * Spring mobile
 
* JSON Web Tokens
* [Google Drive REST API](https://developers.google.com/drive/v3/web/about-sdk)
* [JSONDoc](http://jsondoc.org/)
* Mybatis
* Mysql
